# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in One Terminal, please report it responsibly:

1. **DO NOT** create a public GitHub issue
2. Email us at **fanapplyforu@gmail.com** with:
   - Description of the vulnerability
   - Steps to reproduce
   - Potential impact
3. We will acknowledge receipt within 48 hours
4. We will provide a fix timeline within 7 days

## Scope

- One Terminal desktop application
- One Terminal web platform (oneterminal.zeabur.app)
- Community API endpoints

## Data Privacy

- All user data is stored locally on the user's device
- API keys never leave the user's machine
- Direct messages are ephemeral — deleted from server upon delivery
- No telemetry, no tracking, no analytics
